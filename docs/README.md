## 更新方式

THUAI8 文档不在此仓库内更新，可以在这里存放文档临时版本

### 更新方式：
- fork [eesast/docs](https://github.com/eesast/docs) 到自己仓库
- **每次更新**都建立一个临时的分支用于更改
- 在新的分支上进行文档更新
- 向 [eesast/docs](https://github.com/eesast/docs) master分支提pull request
- 删除本地临时分支
